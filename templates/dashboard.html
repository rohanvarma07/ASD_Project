Getting Started

📋 How to Use This System

Step 1: Prepare Your Dataset

Ensure your CSV file contains the following columns:

  • ID - Unique identifier for each record
  • A1_Score through A10_Score - Screening question responses (0 or 1)
  • age - Age of the individual
  • gender - Gender (m/f or male/female)
  • ethnicity - Ethnic background

Step 2: Upload Your File

Click on "Upload Dataset" and select your CSV file. The system accepts only CSV format files.

Step 3: Preview and Confirm

Review the data preview to ensure your file was uploaded correctly.

Step 4: Automatic Processing

The system will automatically run the machine learning model and generate predictions.

Step 5: View Results

Access comprehensive results including detection statistics and visualizations.
