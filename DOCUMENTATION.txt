## MACHINE LEARNING MODEL

### CURRENT IMPLEMENTATION (Rule-Based)
```python
def predict_asd(df):
    total_scores = df[A1-A10 columns].sum(axis=1)
    predictions = (total_scores >= 6).astype(int)
    return predictions
```

**Logic**: 
- Sum all A1-A10 scores (range: 0-10)
- If sum >= 6: Predict ASD (1)
- If sum < 6: Predict No ASD (0)

### UPGRADING TO REAL ML MODEL
To implement a trained model:

1. Train model with scikit-learn
2. Save model using pickle
3. Load model in app.py
4. Replace predict_asd() function

Example:
```python
import pickle
model = pickle.load(open('models/asd_model.pkl', 'rb'))
predictions = model.predict(X)
```

## CUSTOMIZATION

### Change Color Scheme
Edit `static/css/styles.css`:
```css
:root {
    --primary-color: #4A90E2;  /* Change this */
    --secondary-color: #7B68EE; /* Change this */
}
```

### Change Port
Edit `app.py`:
```python
app.run(debug=True, port=5001)  # Change port
```

### Add Database
Replace in-memory storage with SQLite/PostgreSQL:
```python
from flask_sqlalchemy import SQLAlchemy
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///asd.db'
db = SQLAlchemy(app)
```
