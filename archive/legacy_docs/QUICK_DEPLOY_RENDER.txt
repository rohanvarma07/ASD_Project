Follow these simple steps to deploy your ASD Detection System to the web!

═══════════════════════════════════════════════════════════════════════

STEP 1: GENERATE SECRET KEY (30 seconds)
═══════════════════════════════════════════════════════════════════════

Run this command in your terminal:

```bash
python3 -c 'import secrets; print(secrets.token_hex(32))'
```

📋 COPY THE OUTPUT - You'll need it in Step 5!

Example output: 
a1b2c3d4e5f6...  (64 characters)

═══════════════════════════════════════════════════════════════════════

STEP 2: GO TO RENDER (1 minute)
═══════════════════════════════════════════════════════════════════════

1. Open browser: https://render.com
2. Click "Get Started for Free"
3. Sign up with GitHub account
4. Authorize Render

═══════════════════════════════════════════════════════════════════════

STEP 3: CREATE WEB SERVICE (1 minute)
═══════════════════════════════════════════════════════════════════════

1. Click "New +" button (top right)
2. Select "Web Service"
3. Click "Connect GitHub" if not connected
4. Find repository: "rohanvarma07/ASD_Project"
5. Click "Connect"

═══════════════════════════════════════════════════════════════════════

STEP 4: CONFIGURE SERVICE (2 minutes)
═══════════════════════════════════════════════════════════════════════

Fill in these fields:

┌─────────────────────────────────────────────────────────────────┐
│ Name:           asd-detection-system                            │
│                 (or any name you prefer)                        │
├─────────────────────────────────────────────────────────────────┤
│ Region:         Oregon (US West)                                │
│                 or choose closest to you                        │
├─────────────────────────────────────────────────────────────────┤
│ Branch:         main                                            │
├─────────────────────────────────────────────────────────────────┤
│ Root Directory: (leave blank)                                   │
├─────────────────────────────────────────────────────────────────┤
│ Runtime:        Python 3                                        │
├─────────────────────────────────────────────────────────────────┤
│ Build Command:  pip install -r requirements.txt                 │
├─────────────────────────────────────────────────────────────────┤
│ Start Command:  gunicorn app:app                                │
├─────────────────────────────────────────────────────────────────┤
│ Instance Type:  Free                                            │
└─────────────────────────────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════════════

STEP 5: ADD ENVIRONMENT VARIABLES (1 minute)
═══════════════════════════════════════════════════════════════════════

Click "Advanced" button, then add these:

┌────────────────────┬──────────────────────────────────────────────┐
│ Key                │ Value                                        │
├────────────────────┼──────────────────────────────────────────────┤
│ SECRET_KEY         │ (paste the key from Step 1)                  │
├────────────────────┼──────────────────────────────────────────────┤
│ DEBUG              │ False                                        │
├────────────────────┼──────────────────────────────────────────────┤
│ PYTHON_VERSION     │ 3.11.0                                       │
└────────────────────┴──────────────────────────────────────────────┘

To add each variable:
1. Click "Add Environment Variable"
2. Enter Key name
3. Enter Value
4. Repeat for all 3 variables

═══════════════════════════════════════════════════════════════════════

STEP 6: DEPLOY! 🚀
═══════════════════════════════════════════════════════════════════════

1. Click "Create Web Service" button at the bottom
2. Wait 3-5 minutes for deployment
3. Watch the build logs (automatic)
4. You'll see: "Your service is live! 🎉"

Your app will be available at:
https://asd-detection-system.onrender.com

(or whatever name you chose in Step 4)

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Build Failed?
→ Check logs tab in Render dashboard
→ Verify requirements.txt has all dependencies
→ Ensure Python version is correct

App Not Loading?
→ Check if service is "Live" in dashboard
→ View logs for errors
→ Verify environment variables are set

Static Files Not Loading?
→ Render automatically serves /static/
→ Check browser console for errors
→ Verify file paths in HTML templates

OPTIONAL: ADD CUSTOM DOMAIN
═══════════════════════════════════════════════════════════════════════

If you own a domain:

1. In Render dashboard, go to Settings
2. Scroll to "Custom Domain"
3. Click "Add Custom Domain"
4. Enter your domain (e.g., asd-detector.com)
5. Copy the DNS records Render provides
6. Add records to your domain registrar
7. Wait for DNS propagation (up to 48 hours)
8. Render will auto-configure HTTPS!
